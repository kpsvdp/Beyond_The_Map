Include Switzerland in the Italy file

Two Swiss nights do not make Switzerland the main destination for this route.

Do not apply separately

Submit through Italy and include every Swiss booking in the same application.

Swiss proof to add

Interlaken and Zurich

  • Milan–Interlaken rail reservation and all connection details.
  • Backpackers Villa confirmation for 2 nights.

  • Lauterbrunnen / Grindelwald / Bachalpsee day plan.
  • Interlaken–Zurich and Zurich–Milan rail proof.

If the itinerary changes
